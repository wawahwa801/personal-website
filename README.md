# Jonas Huang - Personal Portfolio

A modern, responsive personal portfolio website built with React showcasing Jonas Huang's education, experience, projects, skills, and achievements.

## Features

- **Modern Design**: Clean, professional layout with gradient header
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive**: Hover effects and smooth transitions
- **Well-Organized**: Clear sections for different types of information
- **Contact Integration**: Direct links to email, phone, and GitHub

## Sections

- **Header**: Name, contact information, and address
- **Education**: Current education at Western Canada High School
- **Experience**: Volunteer work and employment history
- **Projects**: Programming projects and game development
- **Technical Skills**: Programming languages and technologies
- **Awards/Achievements**: Academic and athletic accomplishments

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone or download this repository
2. Navigate to the project directory:
   ```bash
   cd jonas-huang-portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

To create a production build:

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Technologies Used

- **React 18**: Modern React with hooks
- **CSS3**: Custom styling with Flexbox and Grid
- **Google Fonts**: Inter font family for modern typography
- **SVG Icons**: Scalable vector icons for contact information

## Customization

To customize this portfolio for your own use:

1. Update personal information in the component files
2. Modify the color scheme in `src/index.css`
3. Add or remove sections as needed
4. Update the favicon and meta tags in `public/index.html`
